# LineLabels

Tkinter app in Python 3 to label lines

## Setup
Clone the repo:
- `git clone https://github.com/ericke8/LineLabels.git`

Dependencies:
- `tkinter`
- `PIL` (Pillow)
- `os`
- `sys`
- `platform`

## Usage
- `cd LineLabels`
- `python app.py [directory to images] [optional directory to output]`
